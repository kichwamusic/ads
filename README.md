# Kichwa Music Ads

Este repositorio contiene los archivos necesarios para el sitio web de anuncios de Kichwa Music.

## Archivos

- **index.html**: Página principal del sitio web [https://ads.kichwamusic.com/](https://ads.kichwamusic.com/)
- **app-ads.txt**: Archivo de autorización de anuncios para aplicaciones móviles [https://ads.kichwamusic.com/app-ads.txt](https://ads.kichwamusic.com/app-ads.txt)

## ¿Qué es app-ads.txt?

El archivo `app-ads.txt` es un estándar de la industria que permite a los desarrolladores de aplicaciones declarar quién está autorizado a vender su inventario de anuncios. Este archivo ayuda a prevenir el fraude publicitario y asegura que los anuncios sean legítimos.

## Uso

Los archivos en este repositorio deben ser desplegados en la raíz del dominio `ads.kichwamusic.com` para que funcionen correctamente.

## Despliegue

Este repositorio está configurado para ser desplegado en GitHub Pages u otro servicio de hosting estático. Los archivos deben estar accesibles directamente desde la raíz del dominio.
